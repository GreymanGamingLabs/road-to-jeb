# 🐑 The Road to Jeb

A Minecraft-themed quiz web app designed to accompany **The Road to Jeb** paper board game!

## 🎮 How to Play
1. Set up your **A4 board game** with up to 3 players.
2. When a player lands on a question tile, draw a question using this web app.
3. **Click the card** to reveal the secret Minecraft answer.
4. Click **Next Question** to move to the next card.
5. The first player to reach **Jeb_** wins the game!
